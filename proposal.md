# Problem
Ideas are dirt cheap but extremely expensive to validate.

# Solution
A vote-based webforum that allows users to vote with their wallets and pay for automated coding processes.

# Explanation
Ideation is a trivial process -- any contemporary model can generate an almost unlimited quantity of ideas
at all levels of complexity. Implementation is still extremely expensive and time consuming, and cannot be
done for free. A vote-with-your-wallet mechanism ties implementation directly to available resources and
sidesteps any questions of validity -- if it's paid for, it gets built.
